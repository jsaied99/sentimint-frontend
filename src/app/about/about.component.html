About

Welcome to Sentimint!!! This project was created for Capstone 2 at the University of Missouri - Columbia. This application allows a user to search through the Twitter API by inputting a keyword. Our website then reports back some interesting statistics drawn from tweets containing that keyword. The user can also sign in to see their history of searches and the stats that resulted from those searches.

We hope you enjoy using it!
