About

The What

This application allows a user to search through the Twitter API for a given key word inputted by a user. The user would also be able to sign in to see their history of searches.

The Why

The team wanted to be able to utilize concepts learned throughout the Web Dev II course as well as be able to utilize Twitter's API since that is something that is not covered in the course.
